Fade Project Overview

Introduction

For this project, I designed a system to simulate a traffic light using Arduino. The green LED stays lit by default, representing a "go" signal. When a button is pressed, the system initiates a sequence that transitions the traffic light through green, yellow, and red, with each LED fading smoothly before moving to the next. Once the red light fades out, the system resets to green and waits for the next button press.

The goals of this project were to:

  • Use digitalRead() to detect button presses.
  • Control LEDs with analogWrite() to create a fading effect.
  • Use a for loop for smooth transitions.
  • Ensure safe current levels with appropriate resistor calculations.

Schematic

Resistor Calculations:

  • LED Resistors (220Ω): Calculated using Ohm’s Law: R = (Vs - Vf) / I, where:
    • Vs: Arduino supply voltage = 5V
    • Vf: Forward voltage for red, yellow, and green LEDs = 1.8V
    • I: Desired current = 20mA (0.02A)
    Resulting in R = (5 - 1.8) / 0.02 = 160Ω. A 220Ω resistor was chosen for additional safety and to slightly reduce the current, prolonging LED lifespan.
  • Button Resistor (10kΩ): A 10kΩ pull-down resistor is used with the button to ensure the pin reads LOW when the button is not pressed. This prevents floating states, ensuring reliable input detection.

The schematic reflects these components and their connections, ensuring the circuit operates safely and effectively. Each LED is connected to a 220Ω resistor in series, and the button uses a 10kΩ pull-down resistor.
